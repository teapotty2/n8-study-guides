Reversible Causes ‚Äî The H's & T's

üü• THE H's (mostly metabolic / volume)
HYPOVOLEMIA ‚ñº
üß† Think if Hx of trauma, GI bleed, AAA rupture, post-op surgery, hemorrhage, dehydration
üîç Check by Look at neck veins (flat = empty tank), check skin turgor, check for blood in drains/abdomen, assess urine output
üëÄ You'd see Flat neck veins, narrow pulse pressure, PEA with fast narrow QRS, tachycardia ‚Üí then pulselessness, cool/clammy skin
üõ†Ô∏è Fix it Volume! Crystalloid/blood wide open through large-bore IVs. Find and stop the source of loss. If surgical ‚Üí get surgeon NOW.
HYPOXIA ‚ñº
üß† Think if Hx of COPD, asthma, CHF, pneumonia, drowning, airway obstruction, sedation/opioids
üîç Check by Check ETT placement with waveform capnography (EtCO‚ÇÇ), listen for bilateral breath sounds, check SpO‚ÇÇ, look for chest rise
üëÄ You'd see Cyanosis, low SpO‚ÇÇ, poor chest rise, absent/unequal breath sounds, no waveform on capnography
üõ†Ô∏è Fix it Secure the airway. Confirm ETT placement, suction if needed, ventilate with 100% O‚ÇÇ. Rule out pneumothorax.

üß† Think if Hx of DKA, renal failure, sepsis, prolonged arrest, lactic acidosis, diarrhea
üîç Check by Get an ABG ‚Äî look at pH. Check basic metabolic panel (BMP), look for anion gap
üëÄ You'd see pH < 7.1, wide QRS on monitor, Kussmaul respirations (before arrest), elevated lactate, anion gap
üõ†Ô∏è Fix it Sodium bicarb 1 mEq/kg IV. Hyperventilate to blow off CO‚ÇÇ. Treat the underlying cause (DKA ‚Üí insulin, sepsis ‚Üí abx).

üß† Think if Hx of renal failure, dialysis, diuretics, crush injury, burns, insulin use, eating disorder
üîç Check by STAT K‚Å∫ level (iSTAT/point-of-care). Look at the ECG pattern ‚Äî it will tell you which direction
üëÄ You'd see HyperK: peaked T waves ‚Üí wide QRS ‚Üí sine wave ‚Üí arrest

üõ†Ô∏è Fix it HyperK ‚Üí Ca¬≤‚Å∫ chloride (stabilize membrane), insulin + D50, bicarb, kayexalate, consider emergent dialysis

üß† Think if Hx of cold exposure, drowning, found outdoors, elderly found down, massive transfusion
üîç Check by Check core temperature (rectal/esophageal ‚Äî not forehead!). Look at ECG for specific changes
üëÄ You'd see Temp < 30¬∞C / 86¬∞F, cold skin, Osborn (J) waves on ECG, stiff muscles, may appear dead ‚Äî "not dead until warm & dead"
üõ†Ô∏è Fix it Active rewarming ‚Äî warm blankets, warm IV fluids, Bair Hugger, warm lavage. Limit defib to 1 attempt if < 30¬∞C, then rewarm first. Hold meds until > 30¬∞C.

üß† Think if Hx of diabetes, insulin/sulfonylurea use, liver failure, sepsis, eating disorder, adrenal insufficiency
üîç Check by Point-of-care glucose ‚Äî one fingerstick, 5 seconds. Should be part of every code workup.
üëÄ You'd see BG < 60 mg/dL, altered mental status before arrest, diaphoresis, seizures, no specific ECG pattern
üõ†Ô∏è Fix it D50 (25g dextrose) IV push ‚Äî one amp. Recheck glucose in 15 min. If no IV: glucagon 1mg IM.
üü¶ THE T's (mostly mechanical obstructions)
TENSION PNEUMOTHORAX ‚ñº
üß† Think if Hx of trauma, central line placement, mechanical ventilation, COPD/asthma (bleb rupture), CPR rib fractures
üîç Check by Auscultate both lung fields ‚Äî listen for absent breath sounds unilaterally. Look at neck veins and trachea. Don't wait for CXR!
üëÄ You'd see Absent breath sounds on one side, tracheal deviation (LATE sign), distended neck veins (JVD), hypotension, difficult to bag
üõ†Ô∏è Fix it Needle decompression ‚Äî 14g needle, 2nd intercostal space, midclavicular line, affected side. Then ‚Üí chest tube. This is a clinical diagnosis ‚Äî treat on exam, don't wait for imaging.
TAMPONADE (Cardiac) ‚ñº
üß† Think if Hx of chest trauma, pericarditis, post-cardiac surgery, cancer (mets to pericardium), renal failure (uremic pericarditis), aortic dissection
üîç Check by Bedside echo/POCUS ‚Äî fastest way to confirm. Look for pericardial fluid and right heart collapse. Check Beck's triad clinically.
üëÄ You'd see Beck's triad: JVD + muffled heart sounds + hypotension. PEA pattern. Distended neck veins (unlike hypovolemia!). Electrical alternans on ECG.
üõ†Ô∏è Fix it Pericardiocentesis (needle drainage) or emergent thoracotomy if traumatic. Needle goes subxiphoid, aimed toward left shoulder. Even removing 20-50 mL can restore perfusion.

üß† Think if Hx of psych meds, opioids, recreational drugs, suicide attempt, accidental ingestion, unknown altered mental status
üîç Check by Look for pill bottles, track marks, patches. Check pupil size (pinpoint = opioids, dilated = stimulants/TCAs). Smell breath. Get tox screen, 12-lead ECG.
üëÄ You'd see Specific ECG patterns: TCA ‚Üí wide QRS + right axis. Opioid ‚Üí bradycardia/apnea. Ca-blocker ‚Üí bradycardia. Digoxin ‚Üí scooped ST. Scene clues: paraphernalia, empty bottles.
üõ†Ô∏è Fix it Antidote if known: naloxone (opioids), bicarb (TCA ‚Äî wide QRS), Ca¬≤‚Å∫ (Ca-blocker), digibind (digoxin), flumazenil (benzos ‚Äî use cautiously), lipid emulsion (local anesthetic toxicity). Call Poison Control.

üß† Think if Hx of post-op/immobile patient, DVT, cancer, OCP use, recent travel, obesity, sudden collapse ‚Äî classic scenario: post-op patient suddenly codes
üîç Check by Bedside echo ‚Äî RV dilation/strain. Check neck veins (distended). Look for swollen leg (DVT source). EtCO‚ÇÇ suddenly drops during CPR.
üëÄ Sudden PEA in post-op/immobile patient. Distended neck veins, RV strain on echo, sudden drop in EtCO‚ÇÇ. Before arrest: sudden dyspnea, pleuritic chest pain, tachycardia. You'd see
üõ†Ô∏è Fix it tPA (fibrinolysis) ‚Äî alteplase 50mg IV bolus during arrest. Or surgical/catheter-directed embolectomy. If giving tPA, continue CPR for at least 60-90 min (don't stop early ‚Äî it takes time to work).

üß† Think if Hx of chest pain before arrest, known CAD, prior MI, cardiac risk factors (age, DM, HTN, smoking, hyperlipidemia, family hx)
üîç Check by Get a 12-lead ECG if obtainable (even during CPR). Look for STEMI ‚Äî ST elevation in contiguous leads. Troponin if time allows.
üëÄ STEMI pattern on 12-lead (if obtainable). Patient may have reported chest pain, diaphoresis, nausea before arrest. Often presents as V-Fib arrest. You'd see
üõ†Ô∏è Fix it PCI (cath lab) ‚Äî can be done during CPR with mechanical CPR device. Consider tPA if PCI not available. Don't delay for ROSC ‚Äî if STEMI + refractory arrest, activate cath lab during the code.

  1. Confirm airway + waveform capnography (covers Hypoxia)
  2. Bilateral breath sounds? If absent ‚Üí needle decompress (covers Tension Pneumo)
  3. IV access + fluid bolus (covers Hypovolemia)
